# Cubejs Serverless Demo Frontend

## Template / Guide

https://react-dashboard.cube.dev/overview-and-analytics-api

## Generate Types / Statements From Schema

```bash
yarn generate-gql
```

## Generate Amplify Exports

```bash
yarn generate-exports
```

## Setup

### Projen

```bash
yarn projen
```

## Run

```bash
yarn start
```
